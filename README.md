# 💇‍♀️ Beauty Salon Management System

A modern **Beauty Salon web application** built to simplify salon operations and improve customer experience.  
The project allows users to explore services, view details, and book appointments through a clean, responsive interface.

---

## 🚀 Live Demo

https://amerakhaled.github.io/Beauty_Salon/

## ✨ Features

### 👩‍🦰 User Features
- Browse available beauty salon services
- View detailed service descriptions
- Book appointments easily
- Smooth navigation between pages
- Fully responsive UI (mobile / tablet / desktop)

### 💼 Admin Features (if applicable)
- Add / edit / delete services
- View and manage bookings
- Organize customer appointments

## Installation & Setup

Follow these steps to run the project locally:

- Clone the repository
```bash
git clone https://github.com/AmeraKhaled/Beauty_Salon.git
```
- Navigate to the project folder
```bash
cd Beauty_Salon
```
- Install dependencies
```bash
npm install
```
-  Start the development server
```bash
npm start
```
Then open:

http://localhost:3000
